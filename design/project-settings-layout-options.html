Project Settings
该属于哪一层？

三套方案的右侧设置内容、横向分区导航、宽度与留白完全一致，只改变左侧导航语义。打开任一方案后，可以切换 General、Members、Kanban 等分区，也可以切换亮色与暗色。

Option A

保留 Project / Service 侧栏

进入 Project Settings 后仍能看到所有 Service，并可直接切换 Service。右侧设置区完全采用 Cluster 的横向导航与单分区页面。

上下文最完整可直接切 Service当前实现方向
打开完整页面
Option B

完整使用 Cluster 的全局侧栏

Project Settings 被视为全局管理页：左侧显示 Projects、Cluster 与最近项目，不显示当前 Project 的 Service 列表。

与 Cluster 最一致全局导航清晰离开 Service 上下文
打开完整页面
Option C

Project 专注管理侧栏

保留 Project 身份、返回入口、版本和登录信息，但进入 Settings 后隐藏 Service 列表，让管理任务保持专注。

Project 语义明确最少干扰需返回后切 Service
打开完整页面
